教师入口为forum_index_teacher.html，学生入口为forum_index_student.html。
以下为支持的功能：

1.使用学生id查看个人信息（姓名显示的是包君泽）
2.使用学生id查看论坛首页，从学生版论坛首页进入精华区。同样不要打开精华区的帖子。


3.使用教师id查看帖子，注意只有第一个帖子能够打开，其余的都没有做相应的帖子详情界面。
4.使用教师id进行回帖，发表后会回到帖子页。
	回帖内容：
		标题留空 不要上传附件
		内容：“给我发邮件请假就可以。”


4.使用教师id发表新帖，发表后会回到发帖后的首页，不要点击刚才发布的帖子链接，因为没有做链接。
	发表新帖内容：
		标题：“期末到了，同学们复习得怎么样了？”
		不要上传附件
		内容：“祝大家考试顺利。”//内容不重要，因为帖子演示时不打开。
	注意！发帖时间写的是2015.01.12 22：10 如果屏录的时候会录到电脑时间，请务必修改这部分让时间看上去正常。

3.使用教师id演示删除和精华区的解除加精，精华区的帖子都没有做链接，不要打开。
